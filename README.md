# 🍰 True Bites Bakery Website

A modern, responsive, and user-friendly bakery website designed for **True Bites**, a bakery located near DIT University, Dehradun. The website provides an engaging online presence with features such as product showcases, custom cake ordering, customer reviews, and contact information.

🔗 **Live Website:** https://true-bites.vercel.app/

---

## 🌟 Features

### 🏠 Home Page
- Attractive hero section
- Smooth scrolling navigation
- Call-to-action buttons
- Modern bakery-themed design

### 🍩 Product Menu
- Categorized bakery products
- Product images, descriptions, and pricing
- Responsive card layout

### ⭐ Best Sellers Section
- Featured bakery items
- Special badges and highlights
- Customer favorites showcase

### 🛒 Online Ordering
- Add to Cart functionality
- Quantity management
- Dynamic price calculation
- Checkout form

### 🎂 Custom Cake Orders
- Custom cake request form
- Flavor and size selection
- Occasion details
- Special message section

### 🖼️ Gallery
- Bakery product showcase
- Responsive image grid
- Attractive visual presentation

### 💬 Customer Reviews
- Testimonials section
- Customer feedback display
- Interactive design

### 📍 Location & Contact
- Bakery address
- Contact details
- Opening hours
- Google Maps integration placeholder

### 📱 Fully Responsive
- Mobile-first design
- Tablet-friendly layout
- Desktop optimization
- Cross-browser compatibility
